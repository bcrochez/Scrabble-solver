Solveur de Scrabble version 1.2(06/02/2014):

Pour utiliser ce programme il faut avoir installé Python 3.
Pour lancer le solveur de Scrabble double-cliquez sur "main.py" ou d'une autre manière grâce à Python 3.

Vous pouvez tirer 7 lettres au hasard en cliquant sur "Faire un nouveau tirage".

Vous pouvez rentrer un tirage à la main en cliquant sur "Faire un tirage à la main" et en écrivant dans la console Python. Vous devez écrire au maximum 15 lettres.

Vous pouvez trouver le meilleur coup du point de vue du score sur le plateau à partir du tirage courant en cliquant sur "Trouver le meilleur coup".

Pour réinitialiser le jeu cliquez sur "Réinitialiser".

Vous pouvez charger un plateau depuis un fichier texte en cliquant sur "Charger un plateau" et en entrant le nom du fichier dans la console. Si le fichier n'existe pas réessayez.

Vous pouvez sauvegarde un plateau en choisissant "Sauvegarder un plateau" et en entrant le nom du fichier dans la console. /!\ Si le fichier existe déjà il sera remplacé /!\

Les plateaux doivent être dans un format spécifique:
Une ligne correspond à une ligne du plateau, une case vide est réprésentée par "_" sinon par la lettre présente sur la case en MAJUSCULE.
Il doit donc y avoir 15 lignes de 15 caractères. A vos risques et périls de ne pas respecter ce format. Vous pouvez mettre des espaces entre les caractères pour une meilleur lisibilité.
/!\ Les scores ne sont pas sauvegardés /!\
